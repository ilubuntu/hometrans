# 取消收藏SPEC

用户可在收藏页、菜谱详情页的菜谱卡片心形按钮上取消收藏（首页、搜索页卡片心形按钮同样可取消）；取消收藏后，该菜谱从收藏列表移除，心形恢复为未收藏状态，各页面同步刷新；若取消后无任何收藏，收藏页改显示空态。

## 场景一

### 场景概述

用户在收藏页点击心形按钮，取消某菜谱收藏

### 场景逻辑步骤

- 1. 用户位于收藏页，列表中某菜谱卡片显示实心心形按钮（表示已收藏）
- 2. 用户点击该心形按钮
- 3. 系统将该菜谱从本地收藏记录中删除
- 4. 该菜谱即时从收藏列表移除
- 5. 该卡片的心形按钮恢复为未收藏状态（空心心形、中性色）
- 6. 取消状态同步反映到其他页面：首页、搜索页该菜谱卡片心形变为未收藏，详情页顶部心形同样变为未收藏

## 场景二

### 场景概述

用户在菜谱详情页点击心形按钮，取消某菜谱收藏

### 场景逻辑步骤

- 1. 用户进入某已收藏菜谱的详情页，页面顶部显示实心心形按钮
- 2. 用户点击该心形按钮
- 3. 系统将该菜谱从本地收藏记录中删除
- 4. 详情页顶部心形立即恢复为未收藏状态
- 5. 该菜谱同步从收藏列表移除，首页、搜索页心形同步变为未收藏

## 场景三

### 场景概述

取消最后一项收藏后，收藏页改显示空态

### 场景逻辑步骤

- 1. 用户收藏页中仅剩一道菜谱
- 2. 用户取消该菜谱收藏（在收藏页、详情页、首页或搜索页任一处）
- 3. 收藏集合由非空变为空
- 4. 收藏页即时改显示空态（图标、空态标题、提示文案与"浏览菜谱"入口）

## 场景四

### 场景概述

心形按钮为切换语义：对未收藏菜谱点击则加入收藏

### 场景逻辑步骤

- 1. 某菜谱当前处于未收藏状态，心形显示为空心心形
- 2. 用户在任一入口（首页/搜索页/收藏页卡片或详情页）点击该心形按钮
- 3. 系统将该菜谱切换为已收藏（加入收藏的完整行为见"收藏菜谱"规约）
- 4. 该入口心形立即变为已收藏状态，其他页面同步刷新

## 场景五

### 场景概述

"取消收藏"的作用范围与不作用范围

### 场景逻辑步骤

- 1. **仅作用于** 以下入口的心形按钮：收藏页菜谱卡片、菜谱详情页顶部、首页菜谱卡片、搜索页菜谱卡片
- 2. **取消状态全应用同步**：在任一入口取消后，所有展示该菜谱心形的页面与收藏列表均同步刷新
- 3. **不删除菜谱本身**：取消收藏仅删除收藏记录，菜谱数据仍保留，可在首页/搜索继续浏览并重新收藏
- 4. **不作用于** 引导页、设置页、食材库页——这些页面不提供取消收藏入口

> [偏差] 需求描述列举"Saved 或详情页"两处取消入口，实际首页、搜索页卡片心形按钮同样可取消收藏（共四处入口），行为是需求的超集。
