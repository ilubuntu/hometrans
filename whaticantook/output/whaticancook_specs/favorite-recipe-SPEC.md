# 收藏菜谱SPEC

用户可在首页、搜索页、收藏页的菜谱卡片上，或菜谱详情页顶部，点击心形按钮将未收藏的菜谱加入收藏；收藏后心形按钮变为已收藏状态，且各页面的收藏状态同步刷新。

## 场景一

### 场景概述

用户在首页菜谱卡片点击心形按钮，将菜谱加入收藏

### 场景逻辑步骤

- 1. 用户位于首页菜谱列表，某菜谱卡片右上角显示空心心形按钮（表示未收藏）
- 2. 用户点击该心形按钮（如"5-Minute Mug Cake"卡片的心形）
- 3. 系统将该菜谱标记为已收藏，并把收藏记录写入本地存储
- 4. 该卡片的心形按钮立即变为已收藏状态：实心心形、主题强调色，并伴随缩放淡入过渡
- 5. 收藏状态同步反映到其他页面：进入该菜谱详情页时，详情页顶部心形同样显示已收藏状态；进入收藏页时，该菜谱出现在收藏列表中

## 场景二

### 场景概述

用户在搜索页菜谱卡片点击心形按钮，将菜谱加入收藏

### 场景逻辑步骤

- 1. 用户位于搜索页，结果列表中某菜谱卡片显示空心心形按钮
- 2. 用户点击该心形按钮
- 3. 系统将该菜谱标记为已收藏并写入本地存储
- 4. 该卡片心形立即变为已收藏状态
- 5. 收藏状态同步反映到首页、详情页与收藏页

## 场景三

### 场景概述

用户在菜谱详情页点击心形按钮，将菜谱加入收藏

### 场景逻辑步骤

- 1. 用户进入某菜谱详情页，页面顶部右侧显示空心心形按钮
- 2. 用户点击该心形按钮
- 3. 系统将该菜谱标记为已收藏并写入本地存储
- 4. 详情页顶部心形立即变为已收藏状态
- 5. 收藏状态同步反映到首页、搜索页与收藏页

## 场景四

### 场景概述

心形按钮为切换语义：对已收藏菜谱点击则取消收藏

### 场景逻辑步骤

- 1. 某菜谱当前处于已收藏状态，心形显示为实心心形
- 2. 用户在任一入口（首页/搜索页/收藏页卡片或详情页）点击该心形按钮
- 3. 系统将该菜谱切换为未收藏（取消收藏的完整行为见"取消收藏"规约）
- 4. 该入口心形立即变为未收藏状态，其他页面同步刷新

## 场景五

### 场景概述

"收藏菜谱"的作用范围与不作用范围

### 场景逻辑步骤

- 1. **仅作用于** 以下入口的心形按钮：首页菜谱卡片、搜索页菜谱卡片、收藏页菜谱卡片、菜谱详情页顶部
- 2. **收藏状态全应用同步**：在任一入口收藏后，所有展示该菜谱心形的页面与收藏列表均同步刷新
- 3. **不作用于** 引导页、设置页、食材库页——这些页面不提供收藏入口
- 4. **同一菜谱最多收藏一次**：每道菜谱在收藏记录中最多存在一条，不会重复

> [偏差] 需求描述列举"首页、搜索页或详情页"三处收藏入口，实际收藏页菜谱卡片的心形按钮亦可切换收藏（第四处入口），行为是需求的超集。
