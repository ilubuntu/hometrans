# SPEC: REQ-005 订阅列表

## 需求概述

订阅页面应展示用户已订阅的播客列表。每个播客项展示标题、封面、新单集/未播放计数等基础信息。支持点击进入播客详情页。页面还应提供排序、筛选、列数调整、标签分组等功能。空状态时有引导提示。

## 原子场景

### 场景 5.1：查看已订阅播客列表（Happy Path）
**触发**: 用户进入订阅页面
**执行步骤**:
1. 页面加载，显示进度指示器
2. 异步从本地数据库读取已订阅播客列表
3. 数据加载完成后隐藏进度指示器
4. 以网格形式展示所有已订阅播客
5. 每个播客卡片显示：封面图片、标题、新单集计数 badge
6. 滚动列表可查看所有订阅

### 场景 5.2：点击播客进入详情（Happy Path）
**触发**: 用户点击列表中某个播客卡片
**执行步骤**:
1. 用户点击播客卡片
2. 导航到该播客的详情页
3. 详情页展示该播客的单集列表和详情信息
4. 返回后回到订阅列表，滚动位置保持不变

### 场景 5.3：新单集计数 badge 显示（Happy Path）
**触发**: 某播客有新单集时查看订阅列表
**执行步骤**:
1. 后台 Feed 更新拉取到新单集
2. 订阅列表对应播客卡片显示新单集计数 badge
3. 计数数字实时更新（无需手动刷新）
4. 查看新单集后计数减少或消失

### 场景 5.4：调整网格列数（Happy Path）
**触发**: 用户希望改变订阅列表的显示密度
**执行步骤**:
1. 用户打开工具栏菜单
2. 选择列数选项（如 2列、3列、4列、5列）
3. 网格列数立即更新
4. 列数偏好保存到本地，下次打开保持

### 场景 5.5：排序功能（Happy Path）
**触发**: 用户希望调整订阅列表排序方式
**执行步骤**:
1. 用户点击工具栏排序入口
2. 弹出排序选项对话框
3. 可选排序方式：按标题、按最近更新、按新单集数等
4. 选择后列表按新排序方式重新排列
5. 排序偏好保存

### 场景 5.6：计数器设置（Happy Path）
**触发**: 用户希望更改计数 badge 的显示规则
**执行步骤**:
1. 用户点击工具栏计数器设置入口
2. 弹出计数器选项对话框
3. 可选计数规则：新单集数、未播放数、无计数等
4. 选择后 badge 显示规则更新

### 场景 5.7：标签分组过滤（Happy Path）
**触发**: 用户已为播客设置标签，希望按标签过滤
**执行步骤**:
1. 页面顶部显示标签横向列表
2. 用户点击某个标签
3. 列表过滤显示该标签下的播客
4. 显示过滤提示信息
5. 点击"全部"或清除过滤恢复显示所有订阅

### 场景 5.8：多选模式与批量操作（Happy Path）
**触发**: 用户长按某个播客卡片
**执行步骤**:
1. 进入多选模式，被长按的播客被选中
2. 底部显示批量操作浮动菜单
3. 用户可点击其他播客继续选择
4. 批量操作包括：添加标签、删除订阅等
5. 完成操作或取消后退出多选模式

### 场景 5.9：播客上下文菜单（Happy Path）
**触发**: 用户长按或右键某个播客卡片
**执行步骤**:
1. 弹出上下文菜单
2. 菜单选项包括：查看详情、设置、取消订阅等
3. 选择操作后执行对应功能

### 场景 5.10：空状态（边界场景）
**触发**: 用户无任何订阅播客时进入订阅页面
**执行步骤**:
1. 列表区域显示空状态视图
2. 空状态提示"还没有订阅的播客"
3. 提供添加播客入口引导
4. 点击入口可进入添加播客页面

### 场景 5.11：下拉刷新（Happy Path）
**触发**: 用户在订阅列表页面下拉
**执行步骤**:
1. 触发下拉刷新动画
2. 后台检查所有订阅播客的更新
3. 有新单集则更新列表和计数
4. 刷新完成后动画停止

### 场景 5.12：添加播客浮动按钮（Happy Path）
**触发**: 用户在订阅列表页面看到浮动添加按钮
**执行步骤**:
1. 页面右下角显示浮动添加按钮
2. 点击按钮进入添加播客页面

### 场景 5.13：滚动位置保持（边界场景）
**触发**: 用户滚动到列表中部后进入某播客详情再返回
**执行步骤**:
1. 用户滚动到列表中部
2. 点击某播客进入详情
3. 返回订阅列表
4. 滚动位置恢复到之前的位置

### 场景 5.14：数据变化实时刷新（Happy Path）
**触发**: 订阅列表数据发生变化（新增/删除订阅、单集状态变化）
**执行步骤**:
1. 应用收到数据变化通知
2. 订阅列表自动刷新
3. 新增订阅出现在列表中
4. 删除的订阅从列表中移除
5. 计数 badge 同步更新

## 验收要点
- 已订阅播客出现在列表中，展示封面、标题、计数
- 点击播客可进入对应详情页
- 支持排序、筛选、列数调整
- 支持标签分组过滤
- 新单集计数 badge 实时更新
- 多选模式支持批量操作
- 空状态有引导提示
- 滚动位置在页面切换间保持
- 数据变化时列表自动刷新
