# 已下载单集SPEC

已下载页集中展示所有下载完成的单集（含已退订节目中的下载），并在顶部展示正在下载的单集，支持按多维度排序、查看下载日志、删除已播下载、单项播放或删除，以及多选批量操作。

## 场景一

### 场景概述

用户进入"已下载"页，系统一次性加载并展示所有已下载完成的单集，并在顶部展示正在下载的单集。

### 场景逻辑步骤

- 1. 用户通过抽屉导航中的"下载"入口进入"已下载页"
- 2. 系统从数据库中读取所有标记为已下载的单集（包含已退订节目中的下载），按当前保存的排序方式一次性全部加载
- 3. 同时读取当前正在下载的单集，置于列表顶部
- 4. 在列表中展示已下载单集；当没有任何已下载单集时，展示"暂无已下载内容"的空状态文案
- 5. 每个已下载单集项展示其下载状态与操作按钮

## 场景二

### 场景概述

用户通过"排序"入口选择已下载单集的排序维度（日期、时长、标题或大小）及升降序，选择后持久化并刷新列表。

### 场景逻辑步骤

- 1. 用户在"已下载页"顶部工具栏点击"排序"
- 2. 弹出排序选择面板，展示可选维度：日期（旧→新 / 新→旧）、时长（短→长 / 长→短）、标题（A→Z / Z→A）、大小（小→大 / 大→小）
- 3. 用户点击某一排序选项（首次选择某维度取默认升降序，再次点击同维度切换升降序）
- 4. 系统将所选排序方式持久化保存，并触发已下载列表刷新

## 场景三

### 场景概述

用户在已下载单集项上通过操作按钮播放或删除该单集下载，按钮行为随设置而异。

### 场景逻辑步骤

- 1. 用户在"已下载页"查看某条已下载单集项
- 2. 该单集项右侧展示操作按钮；按钮行为取决于下载设置：
   - 若设置"下载按钮点击即播放"，则按钮为"播放"，点击直接开始播放该单集
   - 否则按钮为"删除"，点击删除该单集的本地下载（本地节目删除前会给出警告确认）
- 3. 用户也可点击单集进入其详情页进行播放或其他操作

## 场景四

### 场景概述

用户一键删除所有"已下载且已播"的单集下载。

### 场景逻辑步骤

- 1. 用户在"已下载页"顶部工具栏点击"删除已播下载"
- 2. 系统弹出确认对话框，提示将删除所有已下载且已播的单集
- 3. 用户点击"确认"后，系统查询所有"已下载且已播"的单集并删除其本地媒体文件，列表随之刷新；用户也可点击"取消"放弃操作

## 场景五

### 场景概述

用户在"已下载页"查看下载日志。

### 场景逻辑步骤

- 1. 用户在"已下载页"顶部工具栏点击"下载日志"
- 2. 系统弹出下载日志面板，展示历次下载的成功与失败记录
- 3. 用户可查看失败原因等详情后关闭日志面板

## 场景六

### 场景概述

用户在"已下载页"长按单集进入多选模式，对所选单集批量执行操作。

### 场景逻辑步骤

- 1. 用户在"已下载页"长按任一单集
- 2. 系统进入多选模式，浮出悬浮操作菜单，菜单包含可对所选单集批量执行的若干操作（删除、下载、标记已播/未播、加入/移出队列、加入/取消收藏、分享、重置进度等）
- 3. 用户选择若干单集后点击某个操作；若未选中任何单集则提示"未选择任何项"
- 4. 系统对该批所选单集执行操作并写入数据库，完成后退出多选模式

## 场景七

### 场景概述

下载状态变化时，已下载页自动刷新。

### 场景逻辑步骤

- 1. 当某单集下载完成、开始下载或下载被删除时
- 2. 系统更新进行中下载集合与已下载列表：新完成的单集加入列表、已删除的单集移出列表、正在下载的单集显示在顶部
- 3. "已下载页"在打开或再次打开时自动反映最新状态

## 场景八

### 场景概述

"已下载"功能的作用范围与不作用范围。

### 场景逻辑步骤

- 1. **仅作用于** 以下入口进入的"已下载页"：抽屉导航中的"下载"入口
- 2. **不作用于** 以下位置：单集总列表页、收件箱页、收藏页等不限定为"已下载"，也不提供"删除已播下载""下载日志"专属入口；这些页面不展示已下载列表
