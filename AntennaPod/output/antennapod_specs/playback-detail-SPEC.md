# REQ-010 播放页详情 - 需求规格

## 概述
播放详情页是用户播放音频时的完整界面，展示当前播放剧集的完整信息，包括封面、标题、播客名称、进度条、播放速度、睡眠定时器入口，以及章节导航和节目描述。

## 原子场景

### 场景 1：查看播放页完整信息（正常流程）
**触发**：用户从迷你播放器展开进入播放详情页

**执行步骤**：
1. 异步加载当前播放媒体的完整信息（包含元数据和章节）
2. 封面区域显示：
   - 剧集封面图（圆角显示，支持加载章节专属图片）
   - 剧集标题（长按可复制，点击可展开滚动）
   - 播客名称 + 发布日期（点击可跳转到该播客页面）
3. 如果媒体包含章节，显示章节导航按钮（上一章/下一章）
4. 底部进度区域显示：
   - 当前播放位置（考虑播放速度转换后的时间）
   - 总时长或剩余时间（点击可切换显示模式）
   - 进度条（带章节分隔标记）
5. 底部控制栏显示后退/播放/快进/跳过按钮

### 场景 2：在封面页和描述页之间切换
**触发**：用户在播放详情页水平滑动

**执行步骤**：
1. 默认显示封面页（第一页）
2. 向左滑动切换到描述页，显示节目的 show notes / 详细描述
3. 向右滑动切回封面页
4. 切换到描述页时自动滚动到顶部

### 场景 3：拖动进度条查看章节信息
**触发**：用户长按并拖动进度条滑块

**执行步骤**：
1. 拖动开始时显示预览卡片（带缩放动画）
2. 预览卡片显示当前拖动位置对应的时间
3. 如果拖动位置落在某个章节范围内，显示章节标题 + 时间
4. 松手后 seek 到目标位置
5. 如果松手位置在某个章节起点附近，自动跳到该章节起点

### 场景 4：进入睡眠定时器设置
**触发**：用户点击工具栏中的睡眠定时器菜单项

**执行步骤**：
1. 从工具栏菜单点击"设置睡眠定时器"（定时器未激活时）
2. 弹出睡眠定时器设置面板
3. 设置完成后，菜单项切换为"禁用睡眠定时器"（定时器激活时）

### 场景 5：进入播放速度设置
**触发**：用户点击播放页中的速度按钮

**执行步骤**：
1. 点击速度文字按钮
2. 弹出播放速度选择面板
3. 选择速度后，播放页速度文字更新

### 场景 6：查看缓冲状态
**触发**：播放流媒体时正在缓冲

**执行步骤**：
1. 缓冲开始时显示加载进度指示器
2. 缓冲进度反映在进度条的次要进度条上
3. 缓冲结束后隐藏加载指示器
