# REQ-014 睡眠定时器 - 需求规格

## 概述
睡眠定时器允许用户设定一个时间或集数，到达后自动暂停播放。支持按分钟倒计时和按集数停止两种模式。还提供延长定时器、自动启用、摇晃重置、结束震动等辅助功能。

## 原子场景

### 场景 1：设置时间倒计时定时器（正常流程）
**触发**：用户在播放页工具栏中选择"设置睡眠定时器"

**执行步骤**：
1. 弹出睡眠定时器底部面板
2. 定时器类型选择为"分钟"
3. 输入框预填上次使用的分钟数
4. 用户输入或调整分钟数
5. 点击"设置"按钮
6. 验证播放服务正在运行（若无播放则提示"无媒体播放"）
7. 保存设定值到偏好
8. 向播放服务发送定时器指令
9. 面板切换为倒计时显示模式，实时显示剩余时间
10. 播放页工具栏菜单项切换为"禁用睡眠定时器"

### 场景 2：设置按集数停止定时器
**触发**：用户在睡眠定时器面板中切换到"集数"类型

**执行步骤**：
1. 从类型下拉框选择"集数"
2. 输入框显示上次使用的集数
3. 用户输入想要播放的集数
4. 系统检查输入集数是否超过队列剩余集数
5. 若超过：显示提示信息
6. 若未超过或连续播放已开启：允许设置
7. 点击"设置"按钮，发送定时器指令
8. 面板显示剩余集数

### 场景 3：延长正在运行的定时器
**触发**：定时器正在运行时用户点击延长按钮

**执行步骤**：
1. 面板显示三个延长选项（如 +5、+10、+30 分钟/集数）
2. 用户点击某个延长按钮
3. 计算新的定时器结束时间（当前剩余 + 延长值）
4. 向播放服务发送延长指令
5. 面板更新显示新的剩余时间

### 场景 4：手动禁用定时器
**触发**：用户点击"禁用"按钮

**执行步骤**：
1. 用户在面板中点击禁用按钮
2. 向播放服务发送禁用指令
3. 定时器停止倒计时
4. 面板切换回设置模式
5. 播放页工具栏菜单项切换回"设置睡眠定时器"

### 场景 5：定时器自然结束
**触发**：倒计时归零

**执行步骤**：
1. 定时器到达零
2. 播放服务自动暂停播放
3. 若开启了震动选项，设备震动提示
4. 面板切换回设置模式
5. 播放页工具栏菜单项切换回"设置睡眠定时器"

### 场景 6：自动启用定时器
**触发**：用户配置了自动启用时间段

**执行步骤**：
1. 用户勾选"自动启用"选项
2. 设置自动启用的时间范围（如 22:00 - 07:00）
3. 在指定时间段内开始播放时，自动启用定时器
4. 若时间范围覆盖大部分时间且为集数类型，显示确认对话框

### 场景 7：连续播放未开启时的集数限制
**触发**：集数类型定时器且连续播放未开启

**执行步骤**：
1. 检测到连续播放（自动播放下一集）未开启
2. 集数输入框禁用，锁定为 1 集
3. 显示提示信息："连续播放未开启时只能播放 1 集"
4. 提供跳转到播放设置的按钮
