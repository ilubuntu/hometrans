# 收件箱（新单集）SPEC

收件箱集中展示用户所有订阅节目中标记为"新（未播）"的单集，支持按日期或时长排序、滑动移除单集、批量移除，以及一键清空全部新单集（移出收件箱）。

## 场景一

### 场景概述

用户进入"收件箱"页，系统展示所有标记为"新（未播）"的单集，按默认排序分页加载首屏。

### 场景逻辑步骤

- 1. 用户通过抽屉导航中的"收件箱"入口，或首页"收件箱"卡片上的"更多"入口，进入"收件箱页"
- 2. 系统从数据库中读取所有标记为"新（未播）"的单集，按当前保存的排序方式读取第 1 页数据（每页 150 条）
- 3. 同时读取新单集总数
- 4. 在列表中展示新单集；当没有任何新单集时，展示"收件箱为空"的空状态文案

## 场景二

### 场景概述

用户在"收件箱页"向下滚动到底部时，系统自动加载下一页新单集并追加到列表末尾。

### 场景逻辑步骤

- 1. 用户在"收件箱页"持续向下滚动
- 2. 当滚动接近列表底部且仍存在未加载的更多新单集时，系统自动发起下一页加载（每页 150 条）
- 3. 加载过程中列表末尾展示加载占位项；加载完成后追加数据并移除占位项；若返回数据不足一页则不再继续分页加载

## 场景三

### 场景概述

用户通过"排序"入口选择新单集的排序维度（日期或时长）及升降序，选择后持久化并刷新列表。

### 场景逻辑步骤

- 1. 用户在"收件箱页"顶部工具栏点击"排序"
- 2. 弹出排序选择面板，展示可选维度：日期（旧→新 / 新→旧）、时长（短→长 / 长→短）
- 3. 用户点击某一排序选项（首次选择某维度取默认升降序，再次点击同维度切换升降序）
- 4. 系统将所选排序方式持久化保存，并触发明细列表刷新

## 场景四

### 场景概述

用户在"收件箱页"对单个新单集进行滑动移除（标记为非新），并提供撤销。

### 场景逻辑步骤

- 1. 用户在"收件箱页"列表中对某条新单集执行滑动操作
- 2. 系统将该单集从"新"状态改为"非新（已播未完成）"，使其移出收件箱
- 3. 系统提供撤销机会，用户可在撤销期限内恢复该单集为"新"

## 场景五

### 场景概述

用户在多选模式下批量将所选新单集移出收件箱。

### 场景逻辑步骤

- 1. 用户在"收件箱页"长按单集进入多选模式，选择若干新单集
- 2. 用户点击悬浮操作菜单中的"移出收件箱"
- 3. 系统将所选为"新"的单集批量改为"非新"，并展示已移出 N 集的提示

## 场景六

### 场景概述

用户一键清空收件箱（将所有新单集移出收件箱）；若未设置"不再询问"，则弹出确认对话框。

### 场景逻辑步骤

- 1. 用户在"收件箱页"顶部工具栏点击"全部移出收件箱"
- 2. 若用户此前已勾选"不再询问"，系统直接执行清空；否则弹出确认对话框，提示将清空全部新单集，并提供"不再询问"勾选框
- 3. 用户点击"确认"后，系统将所有"新"单集批量改为"非新"，收件箱清空并刷新，同时展示"已清空收件箱"提示；用户也可点击"取消"放弃操作
- 4. 若用户在确认框中勾选了"不再询问"，则下次点击"全部移出收件箱"时不再弹出确认框

## 场景七

### 场景概述

"收件箱（新单集）"功能的作用范围与不作用范围。

### 场景逻辑步骤

- 1. **仅作用于** 以下入口进入的"收件箱页"：抽屉导航中的"收件箱"入口、首页"收件箱"卡片上的"更多"入口
- 2. **不作用于** 以下位置：单集总列表页（不限定为"新单集"，且不具备"全部移出收件箱"专属入口）；首页"收件箱"卡片本身仅展示新单集预览子集
