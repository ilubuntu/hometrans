# REQ-012 快进与后退 - 需求规格

## 概述
用户在播放过程中可以使用快进和后退按钮快速调整播放位置。快进和后退的秒数可通过长按按钮进行配置，配置后立即生效并反映在按钮旁边的秒数标签上。

## 原子场景

### 场景 1：快进播放位置（正常流程）
**触发**：用户在播放详情页点击快进按钮

**执行步骤**：
1. 读取用户配置的快进秒数（默认 30 秒）
2. 计算新位置：当前位置 + 快进秒数
3. 调用播放服务 seek 到新位置
4. 进度条和位置文字更新为新位置
5. 如果新位置越过章节边界，章节信息同步更新

### 场景 2：后退播放位置（正常流程）
**触发**：用户在播放详情页点击后退按钮

**执行步骤**：
1. 读取用户配置的后退秒数（默认 10 秒）
2. 计算新位置：当前位置 - 后退秒数（最小为 0）
3. 调用播放服务 seek 到新位置
4. 进度条和位置文字更新为新位置
5. 如果新位置回到上一个章节，章节信息同步更新

### 场景 3：配置快进秒数
**触发**：用户长按快进按钮

**执行步骤**：
1. 弹出秒数选择对话框，显示可选秒数列表（如 10、15、30、45、60 秒等）
2. 当前选中的秒数高亮显示
3. 用户选择新的秒数值
4. 将新值保存到用户偏好中
5. 快进按钮旁边的秒数标签更新为新值
6. 关闭对话框

### 场景 4：配置后退秒数
**触发**：用户长按后退按钮

**执行步骤**：
1. 弹出秒数选择对话框，显示可选秒数列表
2. 当前选中的秒数高亮显示
3. 用户选择新的秒数值
4. 将新值保存到用户偏好中
5. 后退按钮旁边的秒数标签更新为新值
6. 关闭对话框

### 场景 5：在暂停状态下快进/后退
**触发**：播放处于暂停状态时用户点击快进/后退按钮

**执行步骤**：
1. 快进/后退操作在播放和暂停状态下均可执行
2. 读取配置秒数，计算新位置
3. seek 到新位置
4. 进度条更新，播放保持暂停状态

### 场景 6：快进到接近媒体结尾
**触发**：快进位置超过媒体总时长

**执行步骤**：
1. 计算新位置超出媒体时长
2. 播放器处理到末尾的行为（可能结束播放并跳到下一集）
3. 界面根据播放状态变化更新
